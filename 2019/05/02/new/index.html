重新开始

竟然有点看懂了



这是一次测试.
又想着重新回来写这个博客。

这一段时间感觉混乱，写作也没有坚持下来，
对自己不满意，身体状态也不好。但我觉得即使是混乱中，也不能停止思考自己的状态，跳出来审视这一切。
要不停止学习，思考，才有可能比自己的昨天进步。

又一次发现自己不是一个好学生，因为遇到问题习惯性会恐慌，然后就想着看能不能问问谁，今天再回来看《人人都是工程师》，想着这一次自己不能跑题，要解决不怎么重新开始更新这个博客的问题，一路看到自己学要的这一章，不知道是不是错觉，竟然比之前能理解的多，速度也快。

现在的目标是，继续布置这个博客，围绕它来学习英语，读书并记录思考。
