17年2月份第2周，学习总结（2月06日-2月12日）

2月份第2周用时间总结（2月06日-2月12日）

学习用时间44小时45分钟。

清晨朗读会英语朗读：8小时20分钟，阅读英文书180分钟，（其中《principles》140分钟），语法音标75分钟。天天用英语听课阅读：10小时30分钟。总计用英语大概时间：23小时15分钟。
写作时间：10.5小时。
中文阅读：5小时40分钟。
坐享220分钟。
平板撑21分钟。

比较上周计划 完成了什么？

1. 英语朗读完成，理解，盲听，复习天天用英语完成。语法音标每天听10-20分钟。读原则，争取10分钟搞定。 √ 这一周天天用英语第课程全部听完，后几天第阅读材料理解起来不那么抵触，感觉到自己阅读理解能力在提高。对朗读也采取来调整，更注重盲听和理解。语法音标有时间才能听。
2. 早起，坐享。√ 一周7点前早起，坐享比较认真。
3. 阅读，每天阅读1小时。《黑天鹅》阅读并做读书笔记。阅读完成的不好，黑天鹅只写了一篇读书笔记。
4. 写作。√ 更新了一篇微信公众号，简书2篇，其余是自己的反思日志放在了博客。

5. 平板撑3分钟。√ 每一次可以完成1.5分钟。
6. 抽时间看电影。提醒自己与孩子的关系。√ 这一项完成的比较好，看电影《飞屋环游记》《无敌破坏王》《玩具总动员》。白天大部分时间是在陪玩，外出较多。

怎么调整 有什么情绪？最重要的领悟是什么？



睡眠能做到睡其自然，比如，睡不好也不焦虑，但是早起都做到了，白天抽时间小憩。

这一周的情绪整体是负能量比较多。但是我清楚有情绪的低点是正常的。

可能比较纠结的是，我没有时间学习编程。这一周虽然情绪不高，但是并没有影响我做的routine.
纠结的时候，我就问我自己，什么对我是最重要的，我的生活，我的亲人，我的孩子，这些才是我自己。

学习英语的过程很孤独，但已经不是一次有这体验。因为我担心自己走了弯路，但是最重要的是什么呢？进步的慢一点，也是比起我的开始进步很多了啊。不然阅读体验不会是现在这样。只是有时候听自己的发音，也很害怕，但是目前发音不是最重要，有时间就去反复听音标课程，但应该把大部分的时间放到阅读理解上面。其实要上路很重要，渐行渐修，比如意识到了哪里方法可能不适合自己，就想办法去找更又效率的方法，但是，我自己思考的是，不管什么方法，比如持续积累够那么多公里数，比如阅读，就是要看够那么多才能提高阅读能力。

今天用了1个半小时，把当天的材料看懂了，重新复习了昨天的笔记版本阅读材料。慢慢的找到方法了，也或许是今天的材料相对简单。英语始终是我学习的重点，第一可以直接看原版书，听电影。第二重要的，也是我接下来要重点花时间的是，给还孩子重新买新书，每天朗读起来。

周一开始就发现自己有点情绪低落，因为每天的时间全要陪她们。很焦虑，后来每天都是陪她们出去不同的玩，也发现又有别的收获，比如放风筝的时候看到她们那么开心，陪放了半天的风筝竟发现自己身体很轻松。去和孩子一起体验玩雪孩子很兴奋，自己学会了怎么玩之后很有成就感。看她们那么疯，打开自己很开心，这些都给我触动。

还是总结一下，重要的领悟是什么：人生真相是什么，你会死亡，你会老去，你人生的大部分梦想会落空。要直面并思考，好好去想，怎么才能让自己免于一事无成的老去？

这是从张辉的公众号看到的，也让我不寒而栗。总是往大而美了去说我们成长是为了更好的明天和自己。但是比起大的梦想，先看看脚在哪里，一步步的从养成微习惯开始，或许，长久下来，也会有不一样的意义。

比如，微锻炼，每天给孩子讲个故事，让她含笑睡去。始终平静观察自己，离自己的心最近。

下周计划

1. 英语朗读完成，理解，盲听，复习天天用英语完成。语法音标每天听10-20分钟。读原则，争取10分钟搞定。
2. 早起，坐享。
3. 阅读，每天阅读1小时。
4. 写作。
5. 平板撑3分钟。
6. 每天读故事书，陪她们看电影。学会问问题，像观察一朵花耐心关注她们。

17年1月份第四周，2月份第一周学习总结（1月23日-2月05日）

1月份第四周用时间总结（23日-29日）

学习用时间39小时，50分钟。

清晨朗读会英语朗读：9小时20分钟，阅读英文书160分钟，（其中《principles》140分钟），语法音标30分钟。天天用英语听课阅读：7小时30分钟。总计用英语大概时间：20小时。
写作时间：8.5小时。
中文阅读：6小时40分钟。
坐享210分钟。
平板撑10分钟。

2月份第一周用时间总结（1月30日-2月5日）

学习用时间40小时，45分钟。

清晨朗读会英语朗读：7小时50分钟，阅读英文书180分钟（其中《principles》140分钟），语法音标30分钟。天天用英语听课阅读：8小时10分钟。总计用英语大概时间：19小时30分钟。
写作时间：10小时。
中文阅读：8小时40分钟。
坐享155分钟。
平板撑8分钟。

比较上周计划 完成了什么？

1. 英语朗读完成。天天用英语完成。语法音标每天听30分钟。读原则，争取10分钟搞定。 （用英语基本做到了，原则朗读10分钟搞不定。语法音标没有达标。抽缝隙时间听的，断断续续。天天用英语讲师潇洒毅行给了学习音标的三个阶段的好课程，一是赖世雄，二是American pronunciation workshop和LISA美语，三是American accent training，我找到了第二阶段前一个，听了前三课，感觉比赖世雄音标更直观一点。每天能基本保证朗读不间断，天天用英语过年期间一周只完成了四天。）
2. 早起，坐享，多观察自己。 √ 坐享这两周大部分时间不够专心。反倒是中间时间随时来坐享比较专心。从春节七天早起打卡之后，现在每天做到7点之前起床。

3. 阅读，每天阅读1小时。 √ 这两周阅读完成比较好，因为路上时间比较多，都用来看书，或者听讲座。读完了《卓有成效的管理者》，万维刚的时间简史读书笔记。得到上每天文章留言。《黑天鹅》第一个部分前两章节。听了简七理财讲座，尝试自己去配置买基金份额。听了春节期间新生大学的几个讲座。
4. 写作。 √ 每天写一篇不少于1500字。春节回家几天都在简书上更新，公众号只更新两篇。

5. 平板撑3分钟。完成的不好，春节期间几天没有做平板撑。
6. 抽时间看电影，听古典音乐。留空闲时间，和孩子一起。√ 看了《大空头》，《追梦赤子心》，《教父1》，《风之谷》。古典音乐没有听。白天大部分时间陪孩子玩。

怎么调整 有什么情绪？最重要的领悟是什么？



能做到早起，但是目前我的挑战是，我保证不了晚睡，有时候孩子会很晚睡，白天睡觉时我可以解放出来时间学习，但是她们晚上就很能熬。

尽量做到每一周总结， 这需要好好执行了，如果上一周是有借口，比如没有带电脑，刚好过年，但是接下来就没有什么借口了。17年的1月份时间记录表格，在excle上面统计，没有完成，我总是在赶着完成任务。这是我经常反思自己的，学习英语因为基础太差，觉得到现在还是非常不自信，觉得自己走了弯路。可是我又想起来，我几乎是零的起步，不用够那么多时间，无论是多少好方法，都不会有多大的效果吧，也不能全天时间都用来学习英语。就只有安慰自己，如果这么容易做到，估计也就没有什么价值了。看一看新生大学什么基础的都有，都很认真的把学习英语当作一回事，就可以知道，英语这技能很重要。

不记单词，是短板。现在发现了天天用英语，每天材料阅读和听课，可以用先大致预习，听课，在回头按老师的笔记复习，效果就会好很多。我就又想到了清晨朗读也可以多用这个方法，比如，晚上临睡前把清晨朗读的在读一遍。我还是不能着急，笑来来时公众号推荐的两本英文书，我都下载了样章，发现能看进去，自己看完了三本英文书的样章。

语法音标，很想学习，但是时间不够。接下来还是抽空能听上一小节，比如十分钟就很好了，音标学习一个。

没有阅读什么都没有，写作可以把生活和成长全部穿起来，英语是必须要会的技能。我很纠结，关于编程的报名与学习，实在是没有时间，孩子目前不到上学年龄，而我很大可能要去工作，正在想如果直接通知去工作，孩子怎么办。放弃什么呢？写作？没有写作怎么和自己的心灵对话？英语因为这一次要决心捡起来，所以前面才用去这么多时间。可不可以这样呢？先把英语达到一定的水准再说。也或许有时间，目前我能用来学习的时间只有这么多，比如一天能挤出来5个小时。我其实很想学习编程，比如如果不尝试，怎么会有这个博客。

和孩子的关系，和家人朋友的关系，多想想 我与你，还是我与它。尊重，不要以爱的名义不尊重对方。这一方面我有意识的去提醒自己，因为学习时间总是不能保证会有焦虑情绪，但是意识到最近很少对孩子发脾气，我该怎么做到尊重，我觉得自己的身体健康也非常重要，没有好身体，精神状态也不会好。坐享就是养神，我已经体验到养神对自己健康的帮助。


还是总结一下，重要的领悟是什么：你只能自己先强大，先有能力。靠写作挣钱吗？绝对不是，但是写作是我最好的梳理自己和自己对话的方式，让我知道我在干什么，对我来说最重要的是什么。也是通过写，我知道我的孩子亲人对我最重要，工作对我也很重要，学会理财，要时时刻刻去积累这方面的知识。人会不知觉的就陷入求认同，求认可，求证明的情绪里面去患得失，想一想真的强大，是不需要这些的。

先让自己强大，付出了就对自己要求结果。对这个世界真心。

1. 英语朗读完成，理解，盲听，复习天天用英语完成。语法音标每天听10-20分钟。读原则，争取10分钟搞定。
2. 早起，坐享。
3. 阅读，每天阅读1小时。《黑天鹅》阅读并做读书笔记。
4. 写作。

5. 平板撑3分钟。
6. 抽时间看电影。提醒自己与孩子的关系。
