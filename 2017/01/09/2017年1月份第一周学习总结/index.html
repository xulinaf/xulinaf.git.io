17年1月份第一周学习总结（02日-08日）

总计用时间总结

p>学习用时间45小时。

清晨朗读会英语朗读：10小时20分钟，阅读英文书2小时40分钟《principles》语法音标4小时。天天用英语听课阅读：9小时。总计用英语大概时间：26小时。
写作时间：8小时20分钟。
中文阅读：8小时30分钟。
坐享3小时40分钟。

比较上周计划

1. 英语朗读完成。天天用英语完成。语法音标每天听30分钟。读原则，争取10分钟搞定。 √
2. 早起，坐享，结合我现在的状态，身体已经好转，每天接着早起。 √
3. 阅读，每天阅读1小时。下一周因为是调整计划中，所以暂不增加英语书阅读。 √
4. 写作，我会每天都写，尽量公开，但是公众号，不会更新那么多，争取一周两次。 √
5. 编程，下周计划完成第三遍。 ×
6. 平板撑，每次1分钟，每天两次。还是要求自己完成平板撑。运动要重视起来，照顾好身体。 √
7. 抽时间看电影，留空闲时间，和孩子一起。 √

说明：打√都完成了。阅读，写作，还有用英语，让我没有时间去练习编程。我觉得我需要调整，可能和新年开始，不知道怎么一下适应新的计划有关系。

怎么调整


下周的学习时间不能增加了。
必须写的话，固定在一小时内完成，阅读英语和听课正确控制在一小时40分钟。目前就是天天用英语太占用时间。
阅读保证一天小时。
英语书的阅读暂时还不能开始。
这一周没有开始读新书，还是复习之前读过的书。
每天听语法对我很有帮助，在理解天天用英语材料的时候。
早起做的不好。这样就影响一天的效率。
平板撑每天3分钟，都完成了，有点小小成就感。
每天听20分钟左右的雪峰音乐会，当作调节，这一周完成的比较好。
总是感觉很赶任务，太挑战了，每天必须听课一小时，听课前还要用一小时把材料啃一遍。
年底，家里事情开始多了起来，要分出来一部分时间和精力来。

早起，即使不得已晚睡也好早起。这是我对自己的下一周必须开始的调整。从周二开始。因为今天已经是周一了。
