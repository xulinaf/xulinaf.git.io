17年3月份第1周学习总结（2月27日-3月05日）

3月份第1周用时间总结（2月27日-3月05日）

学习用时间38小时50分钟。

清晨朗读会英语朗读：9小时30分钟，阅读英文书4小时20分钟，其中《principles》大概120，《habits of a happy brain》2小时20分钟。天天用英语听课阅读：6小时40分钟。总计用英语大概时间：20小时30分钟。
写作时间：8小时。
中文阅读：2小时50分钟+70分钟。
坐享200分钟。
跑步4次，每次30分钟。（没有记入总时间）

比较上周计划 完成了什么？

1. 英语朗读完成，理解，盲听，每天晚上复习。天天用英语完成。读原则。每一周写学习英语总结。√ 用英语总时间和上周一样，大概算做每天3小时，出入的地方是朗读时间多了一些，天天用英语时间减少，只学习了四到五课。阅读时间稍微增加。复习做的不好，有2天晚上没有复习。需要写一篇一周阅读，或者朗读这些英语材料的总结，因没有写过，还不知道怎么开始写，但是等一下就先写起来，总结一下上一周的学习材料。

2. 早起，坐享。√ 7天有6天是5点30起床。坐享双盘四次。跑步和双盘不知道是不是一起的缘故，全身疲劳很严重，周末两天都停下来了。

3. 阅读，每天阅读1小时。读完《对生命说是》《学习之道》 没有完成，中文阅读做到不好，对生命说是还差10%看完，学习之道，没有读。

4. 写作。周日晚上停电没有写。其余六天都做到每天不少于1500字。公众号一篇更新，简书2篇。其余是反思日志。
5. 学习看财报。没有完成，是完全没有时间去做这一件事。
6. 平板撑3分钟。每天运动30分钟。√ 虽然没有每天都完成平板撑三分钟或者是运动30分钟，但是相比之前，还是已经做到很不错，打对号，是要继续给自己加油，因为做到很不容易，对我来说。
7. 读故事书，陪她们玩。提醒自己主动。√ 或许学习时间减少，主要是因为春天来了，外出的时间比较多，有两天太晚了没有读故事书，这一周是读英文故事书。

怎么调整 有什么情绪？最重要的领悟是什么？



上一周是36，这一周是38，相差不多。总时间可能自己能利用起来的暂时只能是这样了，如果再接着增多，对生活，还有孩子，就会有很多牺牲，上半年自己的工作计划可能也无法落实，下半年整个家庭计划也会有很大变动，目前就是先能利用多少时间就利用多少时间，好好生活和学习。

这一周做到好的是，能早起，能运动。发现双盘也是锻炼身体的一种方式。对情绪也有很明显的影响。

做的不好的是，学习英语又进入机械模式。中文阅读量不够，看手机时间过多，注意力控制的不够好。

英语学习上面，发现自己还是会患得失一些所谓的更好的方法。比如天天用英语的老师张雷，他写了一篇文章，说一篇文的阅读对于不同的人来说是又选择标准的，比如，1000字的文章生词量不能多于20个，我单是看到这一项就把自己吓住了，因为想一想就知道，我一开始就上来朗读，单词每一篇都是多到自己不敢去数的。

除去最开始对号入座后的恐慌之外，我还是学会了思考。首先，那些方法或许只适合于大部分人，每个人的个人经验和想要达到的目标是不同的。举一个例子就知道了，我入新生大学之后开始学习英语，一开始没有方法，是从一个叫周云虎的人建立的流利说打卡群开始，从少儿英语开始的。后来到200多天时自己放弃了，因为一旦开始了之后我一直在不断的练习，学习中找到更适合我的方法。现在朗读也过去了200多天，再回去看些当初和我一样一起打卡的人，他们可能还是没有读完一本原版书，或者是阅读上面的累积并不明显。我自己肯定也不够好，但是和一起开始的人比起来，我阅读天天用英语的材料，发现没有畏惧， 还是有很多单词要查，还是最好的情况下只能理解大概意思，但是已经有很多不不同了，不是吗？最明显有用的地方，我一开始给孩子买的一系列的英语故事书，当时自己看和理解都头大，现在我很轻松的可以读给她。

什么是很大的进步呢？我能想要现在阅读天天用英语的材料就完全无障碍，也要看自己当初的脚站在哪里。只和自己的昨天比较。另外，自己很真实的感悟，如果清晨朗读材料不是太难，我在盲听的时候能大概听懂意思。还有自己看《habits of a happy brain》这本书的时候，看懂的更多了，那种理解清楚的感觉真的很好。


写作上面，每一次都要总结一下，上一周听了何老师的课，觉得按他说的方法去写故事才是最终的出路。其实我早就开始思考，只是写自己一个人的反思日志对别人来说不具备什么价值，也就是说不具备传播的价值。一个人自己先没有践行，先把一些道理说出来以为自己理解了也是不对的。无论怎么样，写作可以先是对自己最有益处的事情，即使只能写一个人的反思日志也好，也要先坚持写。但是既然想要成长，就肯定会越来越好，自己能琢磨出适合自己的方法，并依照这个方法让自己受益了，我觉得就可以写出来，就有传播价值。说到底，还是要写下去，只是觉得没有转播价值的时候，就默默自己成长。

这几天看胡饮元的公众号，人家不但写，还把自己大概琢磨明白但中国当下政策整理了一下，开始要大干了。说到底我还是觉得，应该写自己的东西，应该写让自己受益的东西。追求挣钱，有欲望一点都没有错。记得连岳也表达过。


一个重要的领悟：人有很多局限，每个人都一样，最大的敌人其实是自己心里总是吓自己，给自己判断说这不行，那有问题的，那个小我。正在尝试一个小方法，从《对生命说是》这本书学来的，做个体验者，对自己的一切说是。那个小我就是由头脑控制的，要做个知行合一的人，首先就是沉浸和做个体验者。先接受你就是当下这个满是问题，不完美的自己，然后慢慢引导头脑，让自己越来越好一点，但是始终不忘记，要先做个体验者。生命是由每一个流逝过去的片段和每一个当下的片段组成的，真正活过就是感知，沉浸，认真体验当下的片段。

还有无论你多想要改变，多想要变成“谁”，都应该提醒自己：生命是有限的。是有尽头的。一个人的时间是有限的。这样一切选择，一切纠结，或许在问自己内心的时候就不一样。


接下来先说一下自己的调整：

早起和坐享，坐享时间不少于每次30分钟。双盘可以每天尝试20分钟之内。早起跑步要一起来，每天5点半早起，跑步可以一周4到5次。


英语。朗读还是要继续，会考虑如果有太难的 ，可能会读两段，尽量放慢，模仿，读准。
天天用英语一直纠结于要不要都学习7天的课程，还是舍弃谁的，结果上一周做到不好。这一周要认真，每天天天用英语至少一小时。总结要写，即使很简单，只是大杂烩，也要先写出来。
背单词抽空每天只背10分钟。语法音标有空就做，尝试写句子。
自己读英文书和给孩子读英文书每天每天都要做。

写作不管发不发表，都要坚持每天反思日志不少于1500字，阅读要放到写作的前面。已经买了印象笔记的高级会员，要认真对待素材这件事。每天收集一个就好，或者是类比，或者是例子，或者是理论。

找一家a股，开始研究财务报表。用学习来的那五个方法。上一周没有去做这个，但是这一周还是放下来，用2到3个小时去做件事情，因为我觉得这件事还是很重要。

陪孩子，让孩子知道，无论她做什么，或者不做什么，我们都爱她，就是因为她是她。发现老大有时候总是在说话的时候考虑大人的想法。和最重要的人一起成长，还是就是我最重要的人，我要弯腰下来。有时候反思老大和老二的性格不同，就是陪老大的时候自己太小心，总是替她做判断，而老二，我放手的时候很多，自己一个人玩什么都敢去亲自尝试。

下周计划

1. 英语朗读完成，理解，盲听，每天晚上复习。天天用英语完成。读原则。每一周写学习英语总结。每天用英语时间不少于3小时。
2. 早起，坐享。早起是5点半，坐享不少于30分钟。做个体验者，及时主动发现自己的情绪，让自己有能量。
3. 阅读，每天阅读1小时。继续读完《对生命说是》《学习之道》。
4. 写作。收集素材，每天收集至少一个素材。
5. 学习看财报。
6. 平板撑3分钟。每天运动30分钟。
7. 读故事书，陪她们玩。观察大自然，无条件爱她们一切。
